# Claude Project Guidance

This is the static website for US Fellows, a national civic fellowship society owned and stewarded by US Council.

Read `NOTES.md`, `readme.md`, and `agents.md` before making changes. `NOTES.md` is the authority for content and positioning.

Key rules:

- Do not restore the retired visa, immigration-pathway, unpaid-placement, USIVA, Tao Learning, Curiosity Research Corporation, or INSTAR Lab narrative.
- Do not invent Fellows, leaders, institutions, statistics, application dates, contact details, awards, or Journal articles.
- Keep the site static, accessible, responsive, and compatible with GitHub Pages.
- Reuse the Astro layouts, components, and shared design system rather than duplicating navigation or adding page-specific frameworks.
- Preserve the Kingster-derived homepage/menu structure and the US Fellows navy/gold/red institutional visual language.
- Treat all participation language as informational until US Council supplies an authorized intake URL or contact endpoint.

Run `npm run validate`, then use `npm run preview` with `npm run smoke` to validate links, assets, browser console output, and desktop/mobile layouts before completing work.
